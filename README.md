# vue3-manage-system

> 🚧 **当前项目正在开发制作中 (Work In Progress)...** 🚧
> 
> 小白前端学习实战，基于 `vue3` + `vite` + `pinia` + `element-plus` + `apifox` + `nvm` + `axios` 实现的后台管理系统。

## 🎯 项目功能与目录规划

根据项目初期设计的思维导图，本项目包含以下核心模块（持续更新迭代中）：

### 1. 项目初始化
- 使用 `vite` 快速构建项目
- 使用 `nvm` 管理 Node.js 版本
- 使用 `nrm` 管理 npm 镜像资源

### 2. Vue-Router 使用
- 路由的基础配置与搭建

### 3. Element-Plus 集成
- 掌握与实践 `element-plus` 的三种使用���式（全局引入、按需引入等）

### 4. Pinia 状态管理
- 使用 `pinia` 替代 Vuex 进行全局状态管理

### 5. Axios 封装与请求管理
- axios 的普通使用方法
- 引入 mock 数据模拟接口
- 学习两种 mock 的使用方式
- 封装 axios 请求/响应拦截器
- 引入与配置三种环境（开发、测试、生产）

### 6. Echarts 数据可视化
- 集成 echarts 进行图表数据展示

### 7. 用户管理
- 实现用户的增、删、改、查 (CRUD)
- 列表分页功能
- 条件搜索功能

### 8. Tag 标签页
- 顶部多标签页签 (TagsView) 的状态与路由联动

### 9. 登录与权限控制
- 登录页面与逻辑实现
- 根据不同角色的权限管理（RBAC 基础）

### 10. 动态路由
- 根据权限动态生成路由表
- 路由菜单信息的持久化存储

### 11. 路由守卫与异常页面
- 配置全局路由守卫拦截
- 统一的 `404` 错误页面跳转

### 12. 面包屑导航
- 动态面包屑 (Breadcrumb) 联动当前路由
